# Production AI Platform Reference Architecture

**Tag:** v1.0.0

This platform is a production-grade AI portfolio demonstrating Tool Gateway architecture, multi-agent orchestration, governance and evaluation frameworks, with machine-readable AI discoverability and 70%+ cost optimization patterns. It is built to run in a real deployment environment (Next.js on Vercel), not as static slides or isolated notebooks. Every capability shown is implemented in code and verifiable end-to-end.

## Key capabilities

- Registry-first Tool Gateway with 13 mapped demos
- Multi-agent orchestration with HITL checkpoint
- Evaluation engine with drift monitoring
- Guardrails and observability with trace-ID propagation
- Governance dashboard with operational metrics
- AI-agent manifest and JSON-LD schema for machine discovery
- RAG-ready resume.md
- Playwright CI with mobile WebKit coverage

Built to reflect the architecture patterns expected of a production AI platform lead.
